# Zhennova - E-commerce de Insumos Informáticos

Sistema de gestión de inventario y ventas para hardware informático. 

## Arquitectura
* **Backend:** Node.js, Express
* **Base de Datos:** PostgreSQL
* **Documentación:** PlantUML (DER y Diagramas de Secuencia)

## Gestión de Proyecto
El proyecto se gestiona mediante metodología Scrum utilizando tableros de Trello.
